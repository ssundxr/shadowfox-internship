# ShadowFox AIML Internship

A collection of Machine Learning tasks completed during the ShadowFox AIML Internship.

## Structure

- **Beginner**: Introductory data analysis and prediction tasks (e.g., Boston House Pricing).
- **Intermediate**: Data exploration and modeling using broader datasets (e.g., SuperStore sales).
- **Advance**: Advanced NLP tasks such as BERT text generation analysis.

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- Hugging Face / Transformers (BERT)
